Address the Smart Home Challenge

SUHAaaS is a pioneering software-as-a-service (SaaS) application that addresses the challenges faced by consumer electronics (CE) manufacturers and broadcasters in delivering software updates and home automation services to the rapidly growing number of IoT devices in homes.

Built without code

The application is built on the Mobius DTaaS Low Code Platform, which utilizes five tools: PI-Pascal Intelligence, BoB-BoltzmannBot, Monet-The Experience Builder, Vinci - The Omnichannel Engagement tool, and HolaVerse - The Marketplace.
